Automated Threat Modelling

Video

A tool that will identify security threats and mitigations based on the components of a project. There were two parts to this project.

  1. Going through the design process of problem statement scoping, analysis of current market solutions, ideation and iterations before
  2. coming up with a novel automated threat modelling tool.

This was a Final Year Project in partnership with DBS. The team comprised of 7 members, with guidance from 2 professors and one industry mentor from DBS.

One of the main features I worked on was the backend logic to generate attack trees representing possible security threats and mitigations based on the components of a project. This was done using Python with Selenium (for scraping info from the threat libraries websites) and Pandas (in handling data).

Fashion Assistant Android App

Github

A fashion app built using Android Studio. The app allows users to store pictures of their existing wardrobe, OOTDs, customise their outfit or generate a random set of outfit based on formality.

This was a course project done by a team of six. The main feature I worked on was the generate outfit page.

Fundraising Blockchain App

Video

A proof-of-concept project which demonstrates how a blockchain fundraising platform can increase transparency and accountability in how a charity handles donations. This is done through tapping on the immutability and transparency properties of the blockchain.
